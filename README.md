# 🎮 GamesReview
 
Site de reviews de jogos feito com Django, inspirado no Metacritic. Usuários podem se cadastrar, explorar jogos e deixar suas avaliações.
 
---
 
## 🚀 Funcionalidades
 
- Cadastro e login de usuários
- Listagem de jogos com capa, gênero e plataforma
- Página individual de cada jogo com descrição completa
- Sistema de reviews com notas de 1 a 10 (estrelas)
- Deleção de jogos (restrita a superusuários)
- Importação automática de jogos via API da RAWG
 
---
 
## 🛠️ Tecnologias
 
- Python 3.12
- Django 6
- Bootstrap 5
- Bootstrap Icons
- API RAWG.io
